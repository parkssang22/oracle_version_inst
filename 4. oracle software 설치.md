## 다음과 같이 에러가 뜨면 설치 중인 창을 끄지 말고 적혀있는 경로로 가서 다음을 수행한다
<img width="796" height="627" alt="12" src="https://github.com/user-attachments/assets/a0e14cdb-bb88-4587-b813-48345f851b5b" />

```bash
[oracle@localhost lib]$ vi ins_ctx.mk

ctxhx: $(CTXHXOBJ) 
         $(LINK_CTXHX) $(CTXHXOBJ) $(INSO_LINK)

# 아래와 같이 수정하고 retry

ctxhx: $(CTXHXOBJ) 
  -static $(LINK_CTXHX) $(CTXHXOBJ) $(INSO_LINK)
```
- -static 을 넣고 수정할 때 꼭 tab키로 띄어야 한다

## 또 에러가 뜨면 다시 한 번 경로로 가서 수정한다
<img width="796" height="624" alt="13" src="https://github.com/user-attachments/assets/10e710d3-d058-4789-a20b-ce0d7d5a3871" />

```bash
[oracle@localhost lib]$ vi ins_emagent.mk

$(SYSMANBIN) emdctl:
              $(MK_EMAGENT_NMECTL)

# 아래와 같이 수정한다

$(SYSMANBIN) emdctl:
              $(MK_EMAGENT_NMECTL) -lnnz11
```
