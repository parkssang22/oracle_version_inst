# oracle_version_inst

1. edelevery에서 버전에 맞는 오라클 소프트웨어를 다운받는다
2. vm에서 환경을 구성한다
3. 리눅스에서 파라미터 및 유저 리소스 설정, 유저 생성, 환경 변수 설정, 권한을 설정한다
4. 다운받은 오라클 소프트웨어의 압축을 풀고 실행 파일을 찾아 실행한다
5. 오라클 소프트웨어를 설치한다(DB를 자동으로 생성할 수 있지만 dbca로 수동 생성했다)
6. 리스너를 생성한다
7. dbca로 데이터베이스를 생성한다
8. 데이터베이스와 리스너를 실행한다
